## Step#3 Create System call sys_hello

```bash
nahid@cseru:~/Desktop$ cd linux-5.15.12/

nahid@cseru:~/Desktop$ mkdir hello && cd hello
nahid@cseru:~/Desktop$ vim hello.c
```

`hello.c`:

```c
#include <linux/kernel.h>
#include <linux/syscalls.h>

asmlinkage long __x64_sys_hello(void) {
        printk("CSE RU is the Best\n");
        return 0;
}
```

```bash
nahid@cseru:~/Desktop$ vim Makefile
```
